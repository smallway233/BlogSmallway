TOMCAT安装与配置

约 264 字小于 1 分钟

Tomcat

2025-02-17

请注意，安装之前请先确保自己拥有JAVA环境

检测Java环境

打开cmd（Win+R，输入cmd，点击回车）

输入

Java -version

若能显示Java版本则为已安装Java

若不能显示Java版本，请先自行安装Java

安装TOMCAT

1.下载TOMCAT安装包

点击此处跳转至我的下载站下载安装包

2.安装TOMCAT

双击安装包运行安装

Ⅰ.当安装包在此处时，选择【FULL】既完整安装

Ⅱ.选择jre环境（选择对应的Jre环境路径）

Ⅲ.选择TOMCAT安装路径

Ⅳ.完成安装

至此，已完成TOMCAT的安装
