我是碎白，一名热衷于摆烂的大学生，同时也是VR音游的忠实爱好者。今天非常激动地向大家宣布，《音游助手》迎来了重大版本更新！作为一款专为VR音游爱好者设计的辅助工具，我一直致力于不断提升大家的游戏体验。而这一次，我倾注了大量心血，为大家带来了全新的【闪韵灵境】曲库功能。

### 新功能亮点

::: tip 亮点
*   **闪韵灵境曲库**：此次更新中，特别加入了闪韵灵境游戏的曲库功能，为VR音游玩家提供了丰富多彩的曲目选择，让您在游戏中畅享更多乐趣。
*   **玩家贡献，精彩无限**：全新的曲库功能不仅仅是软件更新，更是广大玩家智慧与热情的结晶。曲目全部来源于无数热爱闪韵的玩家精心制作的谱子，每一首曲目都凝聚了大家对该游戏的热爱与创意。
*   **流畅体验**：在更新中，对软件的性能进行了全面优化，无论是加载速度还是操作体验，都得到了显著提升，让您的游戏体验更加顺畅与愉快。（修复一个BUG，新增多个BUG）
:::

### 关于我

::: info 关于作者
作为一名在校大学生与软件开发者（bushi，我说我还会唱歌你们信吗），我始终坚持将学习与兴趣相结合。开发《音游助手》不仅仅是一个项目，更是一种对音游文化的热爱与传承。通过这款软件，我希望能为广大VR音游玩家提供一个便捷、高效的辅助工具，让大家能更专注于享受音乐与游戏的乐趣。同时，我也非常感谢每一位为闪韵灵境谱子贡献力量的玩家，是你们的创意和付出让这次更新充满了无限可能。

### 未来展望

::: tip 计划中
此次更新只是一个开始，未来我计划推出更多激动人心的新功能：

*   **玩家自定义谱子上传功能**：让玩家能够自由上传自己制作的谱子，进一步丰富曲库内容。
*   **已有谱子认领功能**：为玩家提供认领已有谱子的机会，未来我们将对谱子实行标签、分类等管理方式，使搜索和筛选更加便捷。
*   **谱子评论功能**：打破玩家与谱师之间的隔阂，让大家可以在评论中交流创意、反馈建议，共同推动音游社区的繁荣发展。
*   **目前Neo3用户依旧无法使用**：因为4和3设备的分辨率不同。导致UI适配不好弄（我学艺不精）所以下一个小版本更新就会是支持Neo3用户的使用！
*   **玩家自定义谱子社区网站**：[音游助手网站](%5B%E9%9F%B3%E6%B8%B8%E5%8A%A9%E6%89%8B%E7%BD%91%E7%AB%99%5D\(https://bs.smallway.top/\)) 目前已经作为半成品上线了，后期上传谱子，认领谱子，评论谱子等功能都可通过该网站实现。
:::
