# 🔔主要功能

::: tip 功能列表
❗直接使用VR终端下载BS官网的曲包，并一键导入至游戏中，实现快速游玩的目的。不需要使用电脑等其他设备导入歌曲。

目前具体功能如下：

*   曲子名搜歌
*   作者名搜歌
*   谱师名搜歌
*   曲子KEY搜歌（代码搜歌）
*   曲包整理（测试中）
*   曲包音频预览
*   玩家推荐曲包下载
*   比赛详情功能
:::

# 🚦使用教程

1.  下载链接为[光团助手 | 碎白的下载站 (smallway.top)](http://yun.smallway.top:5244/%E5%85%89%E5%9B%A2%E5%8A%A9%E6%89%8B) [http://yun.smallway.top:5244/%E5%85%89%E5%9B%A2%E5%8A%A9%E6%89%8B](http://yun.smallway.top:5244/%E5%85%89%E5%9B%A2%E5%8A%A9%E6%89%8B)，复制该链接（VR中访问请直接点击该链接）
2.  打开\[PICO VR\] app![image-20230816135904388](https://smallway.oss-cn-beijing.aliyuncs.com/image-20230816135904388.png)
3.  粘贴复制的链接，点击【分享】![image-20230816140955306](https://smallway.oss-cn-beijing.aliyuncs.com/image-20230816140955306.png)
4.  此时你的VR一体机内就会跳转出软件的下载链接，点击【下载】![image-20230816141124729](https://smallway.oss-cn-beijing.aliyuncs.com/image-20230816141125609.png)![image-20230816141148795](https://smallway.oss-cn-beijing.aliyuncs.com/image-20230816141148795.png)
5.  下载完成后点击【安装】![image-20230816141222691](https://smallway.oss-cn-beijing.aliyuncs.com/image-20230816141222691.png)
6.  安装完成后你就可以在资源库中找到【光团助手-VR】app，点击打开![image-20230816141339257](https://smallway.oss-cn-beijing.aliyuncs.com/image-20230816141339257.png)
7.  此时会弹出需要授权存储权限，请点击【允许】，存储权限仅用于下载曲包。若未出现或点击了拒绝，请看本篇内容底部问题描述。
8.  此时你可以在此次搜索你想玩的歌曲，点击对应的歌曲进入描述页面![image-20230816141556919](https://smallway.oss-cn-beijing.aliyuncs.com/image-20230816141556919.png)
9.  点击【导入至游戏】，曲包就会开始下载，解压，并导入对应目录。

![image-20230816141742514](https://smallway.oss-cn-beijing.aliyuncs.com/image-20230816141742514.png)

## 问题描述

#### Q：我拒绝了存储权限怎么办？

1.长按【光团助手-VR版】，点击【查看权限】

![image-20230816142000588](https://smallway.oss-cn-beijing.aliyuncs.com/image-20230816142000588.png)

2.点击右上角的【查看详情】

![image-20230816142054787](https://smallway.oss-cn-beijing.aliyuncs.com/image-20230816142054787.png)

3.勾选【存储权限】，勾选后，请点击【强制停止】

![image-20230816142127812](https://smallway.oss-cn-beijing.aliyuncs.com/image-20230816142127812.png)

#### Q：我卡在”正在开始准备工作”页面怎么办？

**A：解决办法参照上一条如何给予存储权限，或检查下你有没有安装《光之乐团》**
