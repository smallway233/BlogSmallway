_请注意，安装之前请先确保自己拥有JAVA环境_

## 检测Java环境

打开cmd（Win+R，输入cmd，点击回车）

![image-20250217174436211](https://smallway.oss-cn-beijing.aliyuncs.com/image-20250217174436211.png)

输入

```
Java -version
```

若能显示Java版本则为已安装Java

若不能显示Java版本，请先自行安装Java

![image-20250217174530490](https://smallway.oss-cn-beijing.aliyuncs.com/image-20250217174530490.png)
